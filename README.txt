== Description ==

Places WP is the most popular wordpress plugin to enable places support into your wordpress website/blog. It enables your app to collect users location from HTML based browser's location share property, 
add meta tags to the page header for better geo tagging. 

This plugin store all data into your local database, so you don't have to worry about any external api service or payment. The current version is just a start to journey. 
We will bring in many advanced features to make your app integrated to geo-location system.

The current version is the basic version of the plugin. To have more flexibility you should take a look to our Premium version here. <a href="">Visit Premium Website</a>

<h4>Basic Features</h4>
* HTML 5 based share location.
* Sync GeoIP info with Techunits GeoIP Location API.
* Google Maps based for Location selection, geotagging to Pages & Posts.
* GeoIP Location Log & Stats.
* Create/Update Place Type can be used for Place list categorization.
* Create/Update Place.
* Show list of places into frontend.
* Embed place info to any posts, pages or custom post-type with shortcodes.
* Default Settings manager.
* Custom Templating Supported.
* Custom Marker Selection from list of 100+ markers.
* Widget enabled.
* Cross browser compatible
* Support for online map creation.
* Exclusive Online support via email from the plugin page itself.

<h4>Premium Features</h4>
* Get list of Nearby places.
* Filter posts by nearby places.
* Exclusive Online support via email from the plugin page & Skype Call.

== Installation ==

1. Unzip the package and upload contents '/wp-content/plugins/' directory or else you can directly upload the zip package via Wordpress default plugin installer.
2. Activate the plugin through the 'Plugins' menu in WordPress
